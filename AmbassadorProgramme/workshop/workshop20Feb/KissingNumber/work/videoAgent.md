# AI Agent Videos

A curated list of 5 YouTube videos about AI agents.

---

## 1. An Introduction to AI Agents (for 2025)

- **Channel:** Shaw Talebi
- **Length:** 23:47
- **URL:** https://www.youtube.com/watch?v=ZaY5_ScmiFE

**Summary:** This video provides a beginner-friendly introduction to AI agents, covering what they are, why they matter, and how they differ from simple chatbots. It presents three levels of agency: Level 1 (LLM + Tool Use), Level 2 (LLM Workflows), and Level 3 (LLM in a Loop). The video explains key concepts such as tool use, autonomy, and test-time compute scaling, illustrating how agents bridge the gap between LLMs' "imagination land" and real-world problem solving.

---

## 2. Tips for Building AI Agents

- **Channel:** Anthropic
- **Length:** 18:19
- **URL:** https://www.youtube.com/watch?v=LP5OCa20Zpg

**Summary:** Anthropic team members Barry Zhang, Erik Schluntz, and Alex Albert go behind the scenes on Anthropic's influential blog post "Building Effective Agents." They discuss the distinction between agents and workflows, the anatomy of an effective agent prompt, common pitfalls to avoid, overhyped vs. underhyped aspects of agents, the potential and challenges of coding agents, and practical advice for developers exploring agents in production.

---

## 3. How OpenAI Builds AI Agents That Think and Act

- **Channel:** 730 (featuring Josh Tobin from OpenAI)
- **Length:** ~60 min
- **URL:** https://www.youtube.com/watch?v=qfhU7JH000o

**Summary:** Josh Tobin from OpenAI shares OpenAI's strategies for building AI agents that handle multi-step tasks. The discussion covers Deep Research, the Codex CLI tool, web automation with Operator, and developer-facing agents. It provides insight into how OpenAI thinks about agent architecture, reasoning, and the challenges of deploying agents that can reliably think and act in complex environments.

---

## 4. Building More Effective AI Agents

- **Channel:** Anthropic
- **Length:** ~30 min
- **URL:** https://www.youtube.com/watch?v=uhJJgc-0iTQ

**Summary:** This Anthropic video dives deeper into building production-quality AI agents. It covers multi-agent systems, design patterns such as parallelization and MapReduce, common failure modes, and best practices for making agents more reliable. The video discusses how to use Agent Skills and MCP (Model Context Protocol) servers to extend agent capabilities, offering practical guidance for developers moving from prototypes to robust agent systems.

---

## 5. Full Course (Lessons 1-10): AI Agents for Beginners

- **Channel:** Microsoft Developer
- **Length:** 1:04:27
- **URL:** https://www.youtube.com/watch?v=OhI005_aJkA

**Summary:** A comprehensive 10-lesson course from Microsoft covering the full journey from concept to code. Topics include: what AI agents are, choosing an agent framework, designing good agents, tool use design patterns, agentic RAG (Retrieval-Augmented Generation), building effective agents, planning design patterns, multi-agent systems, how agents improve over time, and deploying agents into production. Code samples and hands-on exercises use Microsoft Semantic Kernel and GitHub Models.
